RPT 상품조회 운영형 PWA v2

아이콘 안정 적용 버전입니다.

업로드할 파일:
- index.html
- products.json
- manifest.json
- service-worker.js
- icons 폴더

적용 방법:
1. GitHub Pages 저장소에 이 압축 파일 내용을 전부 업로드/교체합니다.
2. 1~2분 기다립니다.
3. 기존 홈화면 앱을 삭제합니다.
4. Safari 또는 Chrome에서 사이트에 다시 접속합니다.
5. 홈 화면에 다시 추가합니다.

이후 데이터 수정:
- products.json만 교체하면 됩니다.
- 앱 재설치는 필요 없습니다.


v3 변경사항:
- 모바일 화면 폭을 100% 사용
- 태블릿/큰 화면은 최대 900px까지 확장
- 카드 모서리 20px
- 입력창/콤보박스/버튼 높이 52px
- 캐시 버전 v3로 갱신


v4 변경사항:
- 전체 UI를 약 200% 수준으로 확대
- 작은 스마트폰에서는 자동으로 약간 축소
- 캐시 버전 v4로 갱신


v5 변경사항:
- v4 큰 글씨/큰 버튼 유지
- 앱 전체 폭을 100vw로 변경하여 모바일 화면 가로폭을 꽉 채움
- 캐시 버전 v5로 갱신


v7 변경사항:
- 최종 10열 구조 기준 products.json 생성
  상품번호 / 상품명 / 기본가격 / 분류 / 옵션1 / 옵션2 / 옵션3 / 가격 / 상품코드 / 배번호
- 분류/옵션1/옵션2/옵션3을 그대로 사용
- 선택 내역을 휴대폰 LocalStorage에 자동 저장
- 모바일 선택 내역은 우측창이 아닌 하단 장바구니 카드로 표시
- 초기화 시 선택 내역도 삭제
- 사용 데이터 원본: ★상품코드 데이터(1).xlsx


v8 변경사항:
